# NEOLAiA-HUB

# How to contribute

## Requirements
- Docker
- VSCode (with Database client extension to see data in the DB)

## Installation
1. Clone this repository:
```
git clone https://github.com/isislab-unisa/NEOLAiA-hub.git
```

2. Install dev container in vscode and open project in vscode

3. Re-open project in dev-container (click blue icon at the bottom left of vscode)
4. Configure appropriately the .env file
5. Install npm dependency:
```
cd neolaia-hub
npm install
```
6. Run strapi from the neolaia-hub directory:
```
npm run develop
```
